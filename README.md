# Spanning

Spanning (by Kaseya) is a SaaS backup and recovery platform providing cloud-to-cloud data
protection for Microsoft 365, Google Workspace, and Salesforce. It protects over 24,000
organizations and 2.5 million users with automated daily backups, unlimited on-demand backups,
infinite retention, and granular point-in-time restore. Spanning exposes RESTful APIs for
managing user licenses and exporting backed-up data.
